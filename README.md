# GKRTOS: A Personal Real Time Operating System

[![builds.sr.ht status](https://builds.sr.ht/~gary-kim/gkrtos/commits/master/test.yml.svg)](https://builds.sr.ht/~gary-kim/gkrtos/commits/master/test.yml?)

### Building

```bash
mkdir tmp
cd tmp
cmake ..
make -j $(nproc)
```
